# lovefield Integration

## Heads Up!!
- set window.openDatabase as per evironment and let lovefield `WebSQL` to use it.

## Database

### Connection
- name = 'books'
- description = 'Books database'
- version = '1.0.0'
- size = 4 * 1024 * 1024
- storeType = $storeProvider.IndexDB
